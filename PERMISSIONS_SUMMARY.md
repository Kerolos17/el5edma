# ملخص صلاحيات النظام - Servant Permissions

تم تطبيق نظام صلاحيات شامل على جميع الـ Resources في النظام.

## 📋 القاعدة العامة

### الخادم (servant):
- ✅ **يشاهد** البيانات المخصصة له فقط
- ✅ **يحمل** الملفات والتقارير PDF
- ✅ **يستخدم** WhatsApp للتواصل
- ✅ **يستطيع إنشاء** الزيارات (Visits) وطلبات الصلاة (Prayer Requests) فقط
- ❌ **لا يستطيع** إنشاء المخدومين أو الزيارات المجدولة أو الملفات الطبية
- ❌ **لا يستطيع** تعديل أي بيانات (حتى الزيارات وطلبات الصلاة)
- ❌ **لا يستطيع** حذف أي سجلات

### أمين الأسرة (family_leader):
- ✅ إنشاء وتعديل وحذف (ضمن أسرته فقط)

### أمين الخدمة (service_leader):
- ✅ إنشاء وتعديل وحذف (كل الأسر)

### مدير النظام (super_admin):
- ✅ صلاحيات كاملة

## 🛠️ الملفات المساعدة

### PermissionHelper
**الملف**: `app/Helpers/PermissionHelper.php`

دالة مساعدة مركزية للتحقق من الصلاحيات:

```php
PermissionHelper::canModify()  // هل يستطيع التعديل؟
PermissionHelper::isServant()  // هل هو خادم؟
PermissionHelper::isSuperAdmin()  // هل هو مدير نظام؟
```

## 📝 ملاحظات مهمة

1. **Data Scoping**: كل دور يرى البيانات المخصصة له فقط (محدد في `getEloquentQuery()`)
2. **Button Visibility**: الأزرار تختفي تلقائياً بناءً على الصلاحيات
3. **Direct URL Access**: لو حاول الخادم الوصول للرابط مباشرة → خطأ 403 Forbidden
4. **PDF Downloads**: الخادم يستطيع تحميل التقارير PDF
5. **WhatsApp**: الخادم يستطيع استخدام أزرار WhatsApp للتواصل
